Hier lernst du margin, padding und border kennen

In dieser Aufgabe geht es darum zu verstehen wie sich margin, padding und border auf Elemente auswirken.
Der graue Container mit der weißen Border ist das Element das du in dieser Aufgabe beeinflusst.
Er ist innerhalb eines schwarzen Containers positioniert und beinhaltet wiederum einen orangenen Container.
In der Praxis könnte der orangene Container ein Bild oder ein Text sein.
Verändere dabei nur die Zahlenwerte innerhalb der Spielwiese..

Aufgabe:

  1. kopiere dir folgenden Code in die Datei margin_padding_border.html 
            
<!DOCTYPE html>
<html lang="">
<head>
    <meta charset="UTF-8">
    <style>
        body {
            display: grid;
            place-items: center;
            height: 100vh;
            margin: 0;
            background: linear-gradient(to bottom right, black, white);
        }
        #container {
            background-color: black;
            position: fixed;
            width: 600px;
            height: 600px;
        }
        #content {
            background-color: coral;
            width: 10px;
            height: 10px;
        }
        #item {
            display: flex;
            justify-content: center;
            align-items: center;
            border: 0px solid white;
            background-color: darkslategrey;
            width: 50px;
            height: 50px;


            /* <<<<<<<<<<<<<Spielwiese>>>>>>>>>>>>> */

            margin-left: 0px;
            margin-top: 0px;
            margin-right: 0px;
            margin-bottom: 0px;

            border: 2px solid white;

            padding-left: 0px;
            padding-top: 0px;
            padding-right: 0px;
            padding-bottom: 0px;


            /* <<<<<<<<<<<<<Spielwiese>>>>>>>>>>>>> */

        }
    </style>
</head>
<body>

<div id="container">
    <div id="item">
        <div id="content"></div>
    </div>
</div>

</body>
</html>
  2. Lasse margin, padding und border an den entsprechenden Stellen so dick werden,
    dass sich das Item (grauer Container) optisch in der unteren linken Ecke befindet
  3. Lasse margin, padding und border an den entsprechenden Stellen so dick werden,
    dass sich der Content (orangener Container) optisch unten in der Mitte des Items ist.
  4. Setze die Border Stärke auf 10px und platziere das Item zentral im schwarzen
    Container und den Content zentral im Item
  5. Gib dem Item eine margin-top und eine padding-top von jeweils 300px.
