Hier lernst du margin, padding und border kennen

In dieser Aufgabe geht es darum zu verstehen wie sich margin, padding und border auf Elemente auswirken.
Der graue Container mit der weißen Border ist das Element das du in dieser Aufgabe beeinflusst.
Er ist innerhalb eines schwarzen Containers positioniert und beinhaltet wiederum einen orangenen Container.
In der Praxis könnte der orangene Container ein Bild oder ein Text sein.
Verändere dabei nur die Zahlenwerte innerhalb der Spielwiese..

Aufgabe:

  1. kopiere dir folgenden Code in die Datei positionierung.html im Ordner Positionierung.
    erstelle diesen Ordner falls notwendig. 
            
<!DOCTYPE html>
<html lang="">
<head>
    <meta charset="UTF-8">
    <style>
        body {
            margin: 0;
            padding: 0;
            height: 2000px;
            width: 2000px;
            background: linear-gradient(to bottom right, black, white);
        }
        #container {  /* Abstand von links */
            background-color: black;
            width: 600px;
            height: 600px;
        }
        #item{
            border: 2px solid white;
            background-color: darkslategrey;
            width: 200px;
            height: 200px;
        }
        #content {
            background-color: coral;
            width: 10px;
            height: 10px;
        }

        /*<<<<<<<<<<<<<<<< Spielwiese >>>>>>>>>>>>>>>>*/>
        #container{
            position: static;   /*default Wert*/
            top: 0px;
            left: 0px;
        }
        #item{
            position: relative; /*relativ zum Parent*/
            top: 200px;
            left: 200px;
        }
        #content{
            position: fixed;    /*Parent ist der Bildschirm selbst*/
            top: 300px;
            left: 300px;
        }
        /*<<<<<<<<<<<<<<<< Spielwiese >>>>>>>>>>>>>>>>*/
    </style>
</head>
<body>
<div id="container">
    <div id="item">
        <div id="content"></div>
    </div>
</div>
</body>
</html>
  2. "position: fixed" ist besonders geeignet für Menüleisten. Scrolle einmal runter und wieder rauf.
  3. Default lässt HTML es nicht zu das Elemente eine andere position erhalten. Alle Elemente sind default mit
    der property "position: static" ausgestattet. Deswegen muss die property "position" irgend einen
    anderen Wert als "static" erhalten.
    Setze zuerst bei #container die Werte "top" und "left" auf 500px um festzustellen das HTML das verhindert.
    Ändere daher den "position" wert bei #container zu "absolut".
  4. Wenn du damit fertig bist die left, top, right, und bottom -properties zu testen,
    mach ein div mit folgenden properties: blau, 50px hoch und breit, position: absolute #z-Container. und füge das neue div unter den bestehenden Code in den body. Positioniere es so das es optisch über dem
    schwarzen container ist. jetzt gib dem #z-container die property z-index: -1.
